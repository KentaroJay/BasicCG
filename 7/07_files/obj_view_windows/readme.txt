Windows版 obj_viewの使い方

[マウスドラッグで使用する場合]
- 表示するOBJファイルをマウスでドラッグして、obj_view.exeにドロップする


[コマンドプロンプトを使用する場合]
- コマンドプロンプトを起動する
- obj_view.exe のあるフォルダまで移動する
 例: cd c:\xxx\yyy\obj_view
- 表示するOBJファイルを指定して起動する
 例: obj_view.exe sample.obj



操作方法
・マウスの左ドラッグで回転
・マウスの右ドラッグでズームイン・ズームアウト
・キーボードの[v]ボタンで、頂点の[表示/非表示]の切り替え
